Setup - Adding SDK to your project


1. Download PayPalHereSDK-1.6.10-release.aar

$ git clone https://github.com/paypal/paypal-here-sdk-android-distribution.git
2. Import PayPalHereSDK-1.6.10-release.aar to your project.
You can find the AAR file under sdk folder of the repo. 



3. Verify build.gradle
Open build.gradle of you app and verify the following entries are there, if not add it. 


                    dependencies {
                        compile project(':PayPalHereSDK-1.6.10-release')
                        compile 'com.google.android.gms:play-services-location:8.1.0'

                        //other entries
                    }
ReBuild the project if you have added these lines.

4. Adding Necessary Permission (AndroidManifest.xml)

<uses-permission android:name="android.permission.INTERNET" />
<uses-permission android:name="android.permission.ACCESS_FINE_LOCATION"/>
You many need to request for enabling Location services, if so, include the following line in your Main Activity class.

ActivityCompat.requestPermissions(this,new String[]{Manifest.permission.ACCESS_FINE_LOCATION}, 1);
