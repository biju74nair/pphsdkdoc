Setup - Adding SDK to your project


1. Add SDK Dependency to build.gradle
Open build.gradle of you app and verify the following entries are there, if not add it. 


    dependencies {
        api('com.paypal.retail:here-sdk-debug:2.0.0@aar'){transitive=true}
        compile 'com.google.android.gms:play-services-location:8.1.0'

        //other entries
    }
ReBuild the project if you have added these lines.

2. Adding Necessary Permission (AndroidManifest.xml)

<uses-permission android:name="android.permission.INTERNET" />
